## 项目简介
这是一个基于 Android 平台的高性能实时目标检测 Demo。项目巧妙地结合了最新的移动端开发技术与端侧 AI 推理框架，实现了在手机上流畅运行 YOLOv8n 模型。
项目的核心目的在于展示如何将 **Jetpack Compose** 构建的现代界面与 **NCNN** 驱动的 C++ 底层推理引擎无缝集成，并通过 **JNI** 进行高效的数据交互。
